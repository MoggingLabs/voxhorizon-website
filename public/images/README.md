# Image assets — what to drop here

Premium photography is the #1 lever for the "$10k" feel. Add the files below to
this folder (`/public/images/`), then reference them with `next/image`
(`src="/images/<name>"`). Optimize each to **< 300 KB** (JPG or WebP).

| File | Used by | Recommended size | Notes |
|---|---|---|---|
| `hero.jpg` | Hero | 1280×1600 (portrait 4:5) | Best single remodel shot — bright, high-end kitchen or bath. |
| `results-1.jpg` … `results-4.jpg` | Results / case studies | 1200×900 (4:3) | Before/after or finished projects tied to each stat. |
| `industry-kitchen-bath.jpg` | Industries + /industries/kitchen-bath | 1200×900 | Signature kitchen/bath project. |
| `industry-roofing.jpg` | Industries + /industries/roofing | 1200×900 | Re-roof / roofing crew on site. |
| `industry-decking.jpg` | Industries + /industries/decking | 1200×900 | Finished deck / outdoor living. |
| `founder.jpg` | About / FounderBio | 800×800 (square) | Clean portrait of Diogo. |
| `og.jpg` | Social share card | 1200×630 | Replaces `og-default.png` for link previews. |

Naming is referenced in `docs/frontend-design-prompts.md`. If you change names,
update the section prompts to match.

**Tip:** shoot/scan in landscape and portrait where possible so sections can crop
to their aspect ratios without distortion. Real before/afters convert best.
