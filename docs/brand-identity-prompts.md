# VoxHorizon — Brand Identity Prompt Pack (from scratch)

Build the **complete** VoxHorizon brand from the ground up in **Claude Design**
(claude.ai/design): strategy, logo system, color, type, visual language, voice,
and every application (web · email · social · sales/offers · ads · print). These
assets become the single source of truth used everywhere.

## How to run it
- claude.ai/design → start a project. Paste **Phase 0** first (seed it in every
  project so everything stays consistent), then run phases in order — each
  phase's decisions feed the next.
- Iterate **broadly in chat** (palette, layout, directions) and **surgically with
  inline comments** (spacing, a curve, a color tweak).
- **Attach references** every phase: brands you admire + (for the "evolve"
  direction) the current VoxHorizon logo.
- **Ask for SVG export** on anything vector (logos, icons, motifs). Build one
  living "brand board" canvas as the reference.

## Decide up front
- Keep the **name** "VoxHorizon."
- Choose your starting point in Phase 1: **evolve** the current sunrise/horizon
  concept, or **explore fresh** directions.

## Dependency → website re-sync
Phases 2–4 (logo, color, type) define the tokens the website + all collateral
use. Once locked, hand me the logo + color + type tokens and I'll **re-skin the
live voxhorizon.com** (Tailwind tokens, fonts, logo) to the new brand. Build it
to work on **dark + light** and **web + print**, scaling from billboard to favicon.

## PHASE 0 — Brand Strategy Foundation (paste FIRST, every project)

```
You are my brand strategist and identity designer. We are building the complete
brand identity for VoxHorizon — from scratch — used across website, email,
social, sales collateral, paid ads, and print. Keep the name "VoxHorizon."

THE BUSINESS: a premium lead-generation / growth-partner agency for established
home-improvement contractors (kitchen & bath remodelers $50K+/mo, plus roofing &
decking). We deliver exclusive, pre-qualified, pre-scheduled projects — not shared
leads. Differentiators: exclusivity, pre-qualified appointments, data + AI driven,
performance/ROI, territory scarcity. Featured across 400+ news outlets.

AUDIENCE: established contractors / business owners — practical, results-driven,
skeptical of "marketing," want trust, proof, and premium partnership (not a vendor).

BRAND PERSONALITY: confident, premium, authoritative, modern, trustworthy, a touch
futuristic ("horizon" = growth and looking forward). NOT loud, gimmicky, or cheap.

CONSTRAINTS the identity must satisfy: works on dark AND light backgrounds; works
in full color, 1-color, and reversed; scales from billboard to 16px favicon;
print-ready (CMYK/Pantone) and web-ready (HEX/RGB); accessible (WCAG AA+).

DELIVERABLE FOR THIS PHASE: a one-page brand strategy brief (positioning
statement, brand promise, 3–5 personality adjectives, audience snapshot, emotional
goal, competitive contrast vs shared-lead platforms like Angi/HomeAdvisor) PLUS a
mood board presenting 3 distinct visual directions (color, type feel, imagery,
motif) for me to react to. Present it on a clean canvas.
```

## PHASE 1 — Identity Direction Exploration (pick one)

```
Using the Phase 0 strategy, design 3 distinct brand DIRECTIONS, each as its own
brand board so I can compare and choose (or blend):

- Direction A — "Evolve the Horizon": refine the current sunrise/horizon concept
  (a circular sun/disc with banded horizon lines, cyan→indigo). Make it more
  premium and ownable.
- Direction B — "Fresh geometric mark": a new abstract/geometric symbol expressing
  growth, exclusivity, momentum (no sunset cliché).
- Direction C — "Editorial wordmark/monogram": type-led identity (custom "VOX
  HORIZON" wordmark and a "V/VH" monogram) with minimal symbol.

For EACH direction show: a name + one-line rationale, a logo concept, a color
palette (5–6 swatches), a type pairing (display + body), a graphic motif, and ONE
quick application mockup (e.g., a website hero or business card) so I can feel it
in context. Keep all three genuinely different. I'll pick one to take forward.
```

## PHASE 2 — Logo System

```
Take the chosen direction and build the full LOGO SYSTEM, presented on both dark
and light backgrounds:
- Primary lockup (symbol + wordmark)
- Horizontal and stacked variants
- Standalone logomark / app icon
- Standalone wordmark
- Monogram + favicon (must read at 16px)
- Monochrome (pure black, pure white) and reversed versions
- 1-color and full-color
Also define: construction/clear-space rules, minimum sizes, and a "don'ts" panel
(don't stretch, recolor, add effects, etc.). Show a small real-world test (favicon
in a browser tab, app icon, embroidered cap). EXPORT as SVG (vector) plus PNG at
1x/2x/3x and a favicon/app-icon set. Keep curves clean and grid-based.
```

---

## PHASE 3 — Color System

```
Design the full COLOR SYSTEM for the chosen brand:
- Primary, secondary, and accent colors
- A neutral ramp (page surfaces + text shades) for BOTH dark and light themes
- Semantic colors (success, warning, error, info)
- The signature gradient(s)
For every color give: a name, HEX, RGB, HSL, and CMYK + nearest Pantone for print.
Define usage ratios (e.g., 60/30/10), recommended pairings, and list WCAG AA/AAA
compliant text-on-background pairs. Present a polished color board, and also output
the tokens three ways: (1) CSS custom properties, (2) a Tailwind theme `colors`
block, (3) a printable swatch sheet. Ensure it looks premium on near-black and on
white.
```

---

## PHASE 4 — Typography System

```
Design the TYPOGRAPHY SYSTEM:
- A display/heading typeface and a body typeface (optional mono for data), with a
  one-line rationale and pairing rules. Prefer typefaces with BOTH web and print
  licenses (Google Fonts or a foundry); note licensing.
- A modular type scale (display → h1…h6 → body → caption) with size, line-height,
  weight, and letter-spacing for each, for web and print.
- Heading vs body usage rules, and EMAIL-SAFE fallbacks (e.g., Arial/Georgia).
Show specimens on dark and light. Output font tokens as CSS + Tailwind
`fontFamily`/scale, plus a one-page type-spec sheet.
```

## PHASE 5 — Visual Language

```
Define the VISUAL LANGUAGE that ties everything together:
- Iconography style (stroke weight, corner radius, grid) — show a starter set of
  ~12 icons (territory, calendar/appointment, growth, shield/exclusivity, AI,
  roof, kitchen, deck, phone, check, star, chart).
- Graphic motifs (e.g., the horizon line, banded gradient, momentum arc) and how
  to use them as accents/dividers/backgrounds.
- Patterns / textures for backgrounds.
- Photography art direction: subjects (real remodel before/afters, crews on site,
  finished kitchens/baths/roofs/decks), framing, and a color-grade/treatment
  recipe so all photos feel on-brand.
- Data-viz style for stats/results (the "$170k in 60 days" type numbers).
Present a visual-language board and export the motifs/icons as SVG.
```

## PHASE 6 — Brand Voice & Messaging

```
Define the BRAND VOICE & MESSAGING:

  example lines. Anchor: confident & premium, proof-led, never hypey/ALL-CAPS.

  100 words).
- Core value props and a naming convention for offers/programs.

  post. Keep it tight and specific.
```

## Deliverables & handoff checklist
- **Logos:** SVG (primary, mark, wordmark, monochrome, reversed) + PNG @1x/2x/3x + favicon/app-icon set
- **Color:** CSS variables + Tailwind tokens + swatch sheet (HEX/RGB/HSL/CMYK/Pantone)
- **Type:** font files/links + CSS/Tailwind tokens + type-spec sheet
- **Visual:** SVG motifs + icon set + photo-treatment recipe
- **Templates:** email / social / sales / ads / deck (source + exports)
- **Brand book:** PDF + share URL
- **→ Hand me the logo + color + type tokens** and I'll re-skin the live website to
  the new brand (drop logos into `/public`, update `tailwind.config.ts`,
  `app/globals.css`, `lib/design-tokens.ts`, and the fonts in `app/layout.tsx`).

## Notes on Claude Design for brand work
- It's a visual/front-end tool: great for boards, logos (export SVG), templates,
  decks, and a brand book. For the final logo, refine the exported SVG (or have a
  designer vector-polish) before mass production.
- Generate options before committing (Phase 1's 3 directions).
- Keep one master "brand board" canvas as the living reference and update it as
  decisions lock.
